# Tic-Tac-Toe Web Application

This project is completed as part of the Prodigy Infotech Web Development Internship. It is an interactive Tic-Tac-Toe game built using HTML, CSS, and JavaScript.

## Project Overview

Tic-Tac-Toe is a simple two-player game where players take turns marking X and O on a 3x3 grid. The player who first places three marks in a row, column, or diagonal wins the game.

In this project, the game logic is implemented using JavaScript, and the interface is designed using HTML and CSS.

## Technologies Used

- HTML
- CSS
- JavaScript

## Features

- Interactive 3x3 game board
- Two-player mode
- Play vs AI mode
- Current turn display
- Win detection
- Draw detection
- Score tracking for Player X, Player O, and Draws
- New round option
- Reset score option
- Responsive user interface

## How to Play

1. Open the game in a browser.
2. Select either 2 Players mode or Play vs AI mode.
3. Player X starts the game.
4. Click on any empty cell to place your mark.
5. The game checks for winner or draw after every move.
6. Click New Round to play again.
7. Click Reset Score to clear the scoreboard.
